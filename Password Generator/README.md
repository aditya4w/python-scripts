# Password Generator

A simple command-line password generator built with Python. Generate strong, customizable passwords by choosing the password length and the types of characters to include.

## Features

- Custom password length
- Include uppercase letters
- Include lowercase letters
- Include numbers
- Include symbols
- Cryptographically secure random generation using Python's `secrets` module
- Input validation and error handling

## Requirements

- Python 3.10+

No external libraries are required.

## Usage

Run the script:

```bash
python password_generator.py
```

The program will:

1. Ask for the desired password length.
2. Ask whether to include:
   - Uppercase letters
   - Lowercase letters
   - Numbers
   - Symbols
3. Generate a secure random password.
4. Display the generated password.

## Built With

- Python
- `secrets` module
- `string` module
